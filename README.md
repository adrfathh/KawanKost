# KawanKost - Kost Finder Application

Aplikasi pencarian kost di Yogyakarta yang dibangun dengan React, Tailwind CSS, dan Vite.

## Fitur Utama

### Untuk Pengguna
- Pencarian kost dengan filter
- Sistem favorit
- Detail kost lengkap
- Kontak langsung via WhatsApp
- Registrasi dan login

### Untuk Admin
- Dashboard admin
- Manajemen data kost
- Manajemen pengguna
- Laporan dan analitik

## Teknologi yang Digunakan

- **Frontend:** React 19
- **Styling:** Tailwind CSS v4 + CSS Modules
- **Routing:** React Router DOM v7
- **Icons:** Lucide React
- **Build Tool:** Vite
